# SQL-Associate Certification

## Overview
This repository documents the work and competencies demonstrated in earning the **Associate SQL Certification** issued by **DataCamp**.  
This certification validates the ability to use SQL effectively for data analysis, with a focus on **Data Management** and **Exploratory Analysis**.  

I was officially certified on **November 30, 2025**, under certificate ID **SQA0017648895175**.  
:contentReference[oaicite:0]{index=0}

The certification ensures that candidates possess the knowledge, skills, and abilities required to analyse data using SQL in real-world business environments.

---

## Competency Domains Assessed

### Data Management
Assesses the ability to clean, transform, and manage data in relational databases. Key skills include:
- Querying and manipulating structured data
- Performing joins, filters, and conditional transformations
- Identifying and resolving data quality issues
- Preparing datasets for analytical workflows

These capabilities were tested through hands-on SQL coding challenges that required executing detailed data cleaning and transformation operations.  
:contentReference[oaicite:1]{index=1}

### Exploratory Analysis
Evaluates the candidate’s ability to explore and derive insights from data using SQL. Responsibilities include:
- Generating descriptive statistics
- Profiling datasets to identify trends and relationships
- Aggregating and grouping data for analytical summaries

This domain was assessed through practical problems using PostgreSQL where insights had to be derived from real datasets.  
:contentReference[oaicite:2]{index=2}

## Certification Process

### 1. Timed Exams
The certification includes timed, adaptive exams designed to test SQL proficiency across a broad set of competencies. These exams evaluate both conceptual understanding and applied problem-solving under time constraints.

**Topics Assessed:**
- SELECT statements and basic querying
- JOIN operations and relational logic
- Aggregations and GROUP BY operations
- Data filtering and sorting
- Subqueries and window functions
- Understanding of relational schema design

These examinations provide a rigorous measurement of SQL competency.  
:contentReference[oaicite:3]{index=3}

---

### 2. Practical Exam Submission
The practical exam requires completing a project using **DataCamp DataLab**, applying SQL to solve a real business problem. This component assesses applied analytical skills and the ability to turn raw data into structured insights.

**Skills Demonstrated:**
- Writing complex SQL queries
- Managing and analysing multi-table datasets
- Extracting insights to support business decision-making
- Applying SQL in a realistic, project-based environment

This project integrates both data management and exploratory analysis competencies.
